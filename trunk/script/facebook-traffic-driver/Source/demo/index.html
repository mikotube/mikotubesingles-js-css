JQuery Facebook Traffic Driver

Description

This plugin will take your Facebook presence to new heights. Its elegant look and realtime stats will make people feel like they are making a difference.

Usage

-----
1. Include the latest copy of Jquery in your tag. Jquery can be found at 'http://www.jquery.com'.
2. Include the 'facebooktrafficdriver.js' in your tag after Jquery.
3. Include the 'facebooktrafficdriver.css' in your .
4. Next, write a JavaScript function that calls the plugin.

Settings
---------

***** url
This is the address to the page that will associated with Facebook. ie: http://facebook.com/ladygaga

***** headline
This should be a punchy headline that introduces your call to action.

***** descr
This should be a one sentence description.

***** stream
This will activate a realtime stream of the current number of likes.

***** required
This will make it mandatory to click the like button in order to view the page.

***** delay
This will activate a timer that will cause the popup to hide itself after it expires. This should be a value from 0-99.
